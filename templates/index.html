How to get a Gmail App Password (if using Gmail)
  1. Open your Google Account → Security → App passwords.
  2. Create a new app password for "Mail" and device "Other".
  3. Copy the generated 16-character password and paste it into the "App password" field above.
  4. Do not share this password or commit it to the repository. It is used only for this upload.
